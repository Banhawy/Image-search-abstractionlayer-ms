Image Search Abstraction Layer Microservice

Use this API to get Image urls and alt tags in JSON format

User Stories

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Example usage:

//Returns image urls and alt tags related to the search term "query" with an offset of 10 items
https://adham-image-search-ms.herokuapp.com/api/search/linux?offset=10

//Returns last 10 serach terms in database
https://adham-image-search-ms.herokuapp.com/latest
